# AT 명령어
VER1.8
1. AT+VERSION =========> 버전
2. AT+NAME\[이름\] ====> 블루투스 모듈 이름 
3. AT+PIN\[번호\] =====> 번호 로 블루투스 핀번호 변경(4자리 숫자)
4. AT+ROLE=\[M/S\] ====> 해당 모듈역할 마스터M 보조S 
5. AT+BAUD\[레이트값\] => 1-1200, 2-2400, 3-4800, 4-9600, 5-19200, 6-38400, 7-57600, 8-115200
